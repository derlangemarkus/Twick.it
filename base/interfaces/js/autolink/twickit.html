Ein Plugin von Twick.it, der Erklärmaschine

Twick.it-Autolink, die twickit-Klasse und der <twick:it>-Tag

Automatische Verlinkung von Textstellen

Mit diesem JavaScript ist es möglich, Begriffe automatisch zu ihrer Erklärung bei Twick.it verlinken zu lassen. Das Skript schaut dazu selbstständig zur Laufzeit nach, ob ein Begriff bereits bei Twick.it erklärt wurde. Ist dies der Fall, wird der Begriff verlinkt. Beim Überfahren des Links erscheint ein Tooltip mit der populärsten Erklärung.

Schritt 1: JavaScript einbauen

Die Verwendung des Skripts ist einfach. Zunächst muss in den <head>-Bereich eurer Seite dieses JavaScript eingebunden werden:
<script type="text/javascript" src="http://twick.it/interfaces/js/autolink/twickit.js"></script>

Schritt 2: Zu verlinkende Begriffe auswählen

Jetzt kann es losgehen. Einfach die zu erklärende Begriffe mit einem beliebigen Tag (z.B. <span>) markieren, das die CSS-Klasse namens "twickit" zugewiesen bekommt. Also so: <span class="twickit">Begriff</span>. Das war's.
Noch ein wenig eleganter wird es mit dem <twick:it>-Tag (ja, mit DOPPELPUNKT). Dazu im öffnenden <html>-Tag zuerst den Namespace definieren. Und zwar mit xmlns:twick="http://twick.it/". Also in etwa so: 
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="de" xmlns:twick="http://twick.it/">
Sobald das gemacht ist, können die zu erklärende Begrife ganz einfach mit <twick:it>Begriff</twick:it> markiert werden.

Schritt 3: Ein wenig mehr Freiheit (Achtung, es wird technisch!)

Wenn ihr wollt, könnt ihr noch ein wenig in die Verlinkung eingreifen. Am besten fügt ihr kurz vor dem schließenden </body>-Tag ein JavaScript-Aufruf ein: 
<script type="text/javascript">
	TwickitAutolink.autolink(
		document.getElementById("bereich"),
		{
			minLength:3
		}
	);
</script>
Die Verlinkung wird durch den Aufruf von InsertTwickitLinks.autolink gestartet. Der Befehl erwartet zwei Parameter.
  1. Ein oder mehrer Knoten des DOM-Baums
  2. Weitere Optionen
Als erster Parameter kann ein einzelnes Element (document.getElementById()) oder mehrere Elemente (document.getElementByTagName(), document.getElementsByClassName()) übergeben werden. Sollen beispielsweise alle Inhalte von <abbr> verlinkt werden, muss document.getElementByTagName("abbr") übergeben werden.

Für die Optionen sind folgende Einstellungen möglich:
  • minLength
    Mindestzeichenlänge, die ein Wort haben muss, damit es bei Twick.it nachgeschlagen wird. Default: 3 Zeichen.
  • disableGeo
    Unterdrückt bei 1 die Verlinkung auf einer Google-Maps-Karte. Default: 0
  • target
    Das Linkziel, das angibt, in welchem Fenster Links geöffnet werden sollen. Default: _blank
  • language
    Das Sprachkürzel (de, en).
